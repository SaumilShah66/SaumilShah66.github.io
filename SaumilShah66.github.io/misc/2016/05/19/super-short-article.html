Numpy

May 19, 2016

Numpy stands for numerical python, and is the best open source module for numerical calculations. You can creat N-dimensional array objects with numpy, which makes it very easy to save and process huge numerical data-sets. Entire backend of numpy has been written in C/C++ and Fortran, which gives amazing processing speed and has number of built in methods or algebra, triginometry and calculus. For more details you can visit official numpy page.

If you have every worked with MATLAB, then it will give you a huge jump start. In many cases you will feel like Numpy is nothing but an open-source version of MATLAB.

Installation

Numpy can be easily installed with python-pip. You can install it with following command.

pip install numpy

Specific version of numpy can be installed with the following command. Please change the version number as per your requirement.

pip install 'numpy==1.18.0'

Creating numpy array

There are multiple ways to create arrays with numpy, each one having their own specific use cases.

Creating arrays from lists and tuples

With array method we can create arrays from python lists. While creating with array method, make sure to include data into square brackets.

>>> import numpy as np
>>> a = [1,2,3]
>>> b = np.array(a)
>>> b
array([1, 2, 3])
>>> c = np.array([3,4,5])
>>> c
array([3, 4, 5])
>>> d = (1,3,5)
>>> e = np.array(d)
>>> e
array([1, 3, 5])

Usually while creating variables in python, we never define any data type, but with numpy you can do it. Numpy arrays can be defined with int16, int32, float32, float64, complex, etc.

>>> a = np.array([1,2,3], dtype = np.float32)
>>> a
array([1., 2., 3.], dtype=float32)
>>> b = np.array([1+1j,2+3j], dtype=complex)
>>> b
array([1.+1.j, 2.+3.j])
